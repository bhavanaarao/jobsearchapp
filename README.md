Jobby App is a prototype designed for searching and filtering content based on ratings and reviews. Utilizing React and React Router, the app provides a dynamic and interactive experience for users seeking relevant content.

Features
Login Route: Secure authentication for accessing personalized content.

Home Route: Overview of available content and search functionalities.

Jobs Route: Browse and filter content based on user-selected criteria such as ratings and reviews.

Job Item Details Route: In-depth view of individual content items with detailed information.

Not Found Route: Custom 404 page for handling unsupported or random paths.

APIs:

login API:https://apis.ccbp.in/login

profile API:https://apis.ccbp.in/profile

jobs API: https://apis.ccbp.in/jobs

jobdetails API: https://apis.ccbp.in/jobs/:id
